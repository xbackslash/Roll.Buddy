ROLL.BUDDY - README



A simple dice and coin rolling app for casual tabletop gaming and fun.



Version: 1.0
Author: @xbackslash
11/28/2025



FEATURES

Roll D4, D6, D8, D10, D12, D20 dice

Flip one or multiple coins

Keeps a running history of rolls and flips

Save history to a text file

Collapsible help panel with instructions

Keyboard shortcuts for quick rolling:

	1–6: Roll corresponding dice (D4–D20)

	Enter: Flip coins

	C: Clear current entry

	A: Clear all history

	S: Save history

	Backspace: Reset die/coin count

	Esc: Close app




HOW TO RUN

Download the Roll.Buddy.exe file.

Double-click to open.

Use the buttons or keyboard shortcuts to roll dice or flip coins.



Notes About Windows Security:

Windows SmartScreen may show a warning:

“Windows protected your PC”

This is normal for apps without a verified digital certificate.

To run the app:

Click More info

Click Run anyway
